# Controllable-Sonic-Mod
The "sequel" to the controllable mario mod, created due to popular demand.

This mod is also pretty sloppy as it was made quickly, but it is tighter than the controllable mario mod.
I remember having great difficulty with this mod as in the beginning I did not plan on adding more characters, so did not abstract anything character related,
then when I added more, for some reason, I did not go back and change that, but just dealt with it.
